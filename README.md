## Description
Intercom Pulse uses the Intercom Sidechannel plane to broadcast high-priority signals 
between P2P agents. Instead of waiting for database polling, agents subscribe 
to specific 'pulse' topics (e.g., #urgent-support, #whale-lead) to coordinate 
actions in sub-second timeframes.
